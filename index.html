HTML Table Reference

Semantic HTML

Tag Name Description
<header> Header a container usually for either navigational links or introductory content
<nav> Navigation used to define a block of navigation links such as menus and tables of contents.
<article> Article represents a self-contained composition in a document
<figure> Figure element represents self-contained content, potentially with an optional caption, which is specified using the <figcaption> element.
<footer> Footer typically contains information about the author of the section, copyright data or links to related documents
